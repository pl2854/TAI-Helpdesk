# Helpdesk for teachers
Created to report computer related issues in classrooms.

Create tickets

Comment on tickets

Created in ReactJS



## Installation
download or clone repo

run: npm install

run: npm start

modify the default.json file with your own MongoDB url.

connect MongoDB

## Usage
Create account with email and password
